# Radio la 35 — Servidor/Backend
Este es el servidor que utilizo el sitio web oficial de la radio de la Esc. Tecnica N°35 Ing. Eduardo Latzina.
Esta pagina contiene: 
- Manejo de cuentas de usuario (Inicio de sesion, Registro de usuario, Administradores)
- Manejo de publicaciones (Eventos, programas, noticias, etc)
- Conexion a una base de datos para el correcto manejo de informacion y datos de los usuarios y otros elementos de la pagina web.
- Forma de contacto con Radio la 35

## Tecnologías utilizadas en Backend:
- Node.js
- Express — para crear el servidor y manejar rutas.
- CORS — para permitir la comunicación segura entre el frontend y el backend.
- Supabase — utilizado como base de datos y servicio de autenticación.
