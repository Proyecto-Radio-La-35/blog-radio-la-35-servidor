# Radio la 35 — Servidor/Backend
Este repositorio contiene el lado servidor que utiliza el sitio web oficial de la radio de la Escuela Técnica N°35 Ing. Eduardo Latzina.
Esta página contiene: 
- Manejo de cuentas de usuario (Inicio de sesión, Registro de usuario, Administradores)
- Manejo de publicaciones (Eventos, programas, noticias, etc)
- Conexión a una base de datos para el correcto manejo y almacenamiento de la información. Lleva datos de usuarios (comunes y administradores), mensajes de contacto, publicaciones y comentarios
- Forma de contacto con Radio La 35

## Tecnologías utilizadas en el backend:
- Node.js
- Express: Crea el servidor y manejar rutas.
- CORS: Permite la comunicación segura entre el frontend y el backend.
- Supabase: Utilizado como base de datos y servicio de autenticación.
